One-liner for converting the like.js file in the "Your Twitter Data" download into plaintext suitable for use with `twarc hydrate` from @documentnow's twarc utility (https://github.com/DocNow/twarc):

sed -e 's/window.YTD.like.part0 = //' like.js | jq -r '.[] | .like.tweetId' > ids.txt
